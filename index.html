06FREQUENTLY ASKED

よくある質問。

兼務でやってる情シスや DX 担当との違いは？ +
兼務情シスは「人間が片手間でやる」前提で、ベンダー比較・契約チェック・社内資料化が後回しになります。Second は同じ仕事を24時間 watching し、見つけた瞬間にドラフトまで仕上げる agent。「あの稟議書、いつ作ろうか」と先延ばしする時間が消えます。
受信箱やデスクトップを見られるのは怖いのですが… +
すべて read-only で、暗号化された上で社内に閉じた環境で処理されます。LLMには匿名化された文脈のみが送信され、再学習目的では利用されません。watching対象（受信箱だけ / デスクトップだけ / 商談だけ）も柔軟に設定可能。GDPR・改正個人情報保護法に準拠。
良さそうなベンダーの商談、Second に任せられますか？ +
はい、proactive vendor scouting モードを使うと、Second が条件にマッチする SaaS を提案し、あなたの代わりに demo を聞いて要約レポートを届けます。営業との一次接触で消える時間がゼロに。気になったものだけ、あなたが本商談に出ればOK。
ベンチマークデータはどこから来ているのですか？ +
匿名化された実契約データ、業界レポート、公開価格表、Second導入企業から共有された契約情報(オプトイン制)、そして毎日 watching している海外 SaaS ニュース（Lenny's / SaaStr / TechCrunch / The Information 等）を組み合わせています。
営業商談に同席する時、相手に「AIが見てる」ことは伝わりますか？ +
いいえ。商談モードでは Second は買い手側のブラウザ/デスクトップで動作し、ベンダー側には一切通知されません。商談の場の空気を壊さない設計です。
料金体系は？ +
初期段階は¥98,000/月のフラット料金で、SaaS購買アドバイス + watching + 業務代行 + vendor scouting すべて込み。年間SaaS購買額に応じたエンタープライズプランも準備中。Vendr型の「削減額未達成時の差額返金保証」も検討中です。
